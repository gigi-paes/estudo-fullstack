Aprendendo HTML e CSS

HTML é uma linguagem semântica, ou seja, devemos usar a tag específica para cada tipo de conteúdo para ser bem interpretado.

A meta tag viewport é usada para que o site se adapte ao tamanho da tela (PC, celular, tablet, etc.).

A estrutura padrão de um site inclui cabeçalho (header), conteúdo principal (main) e rodapé (footer), todos dentro do body.

As tags de título vão de h1 até h6 em ordem de importância.

A tag div é um contêiner genérico usado para agrupar elementos.

Podemos adicionar atributos em cada tag para modificar seu comportamento ou aparência.

No JavaScript, é uma boa prática usar camelCase para nomear funções.

O cabeçalho geralmente contém o logo e links de navegação.

Existem dois tipos de listas em HTML: ordenadas (ol) e não ordenadas (ul).

Para criar links, usamos a tag a (âncora).
